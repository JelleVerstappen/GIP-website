Proces



Eerst keek ik wat voor componenten ik het best zou gebruiken. Ook zocht ik hier op hoe ik ze best kon aansluiten.

Toen maakte ik alle aansluitingen van de joystick en bestuurde ik er 2 motors mee, gebruikmakende van de motordriver(LN298N).

Hierna begon ik aan het regelen via een bluetooth module, tijdens deze fase liep ik soms vast omdat er fouten waren in het programma. Gelukkig heb ik de fouten eruit kunnen halen en werkt alles nu perfect.

Nu dat ik al het regelen en aansturen zo goed als volledig uitgewerkt heb, kan ik beginnen aan het demonteren van de auto en kijken welke componenten waar passen.

Ik ben hier begonnen met het ontwikkelen van mijn app. Dit heb ik gedaan via de App Inventor van M.I.T.(Massachussets Institute of Technology). Deze stuurt data door naar de HC-06, hiermee bestuur je de auto.

Na dat alles werkte ben ik begonnen met het maken van de website. Natuurlijk moet het ook ergens gehost worden, normaal gezien zouden we voor 1 van de 2 sites Github Pages gebruiken. Hierbij heb ik wat problemen mee ondervonden, zoals het linken dat niet werkte zoals gewenst. Daarom host ik beide websites via Netlify.

Ik heb de blog gemaakt met Pelican, deze genereerd website uit kleine tekst bestandjes. Ik maak gebruik van het thema dat meneer T. Issaris gemaakt heeft voor ons. De website heb ik gemaakt met Bootstrap, dit laat je toe om redeljk snel een mooie website te designen.



Elektrisch schema



In het volgend schema kan je alle gebruikte componenten zien en hoe ik ze aansluit, hierdoor kan ik de werking wat makkelijker uitleggen. Zoals je kan zien wordt elk component door een powerbank van stroom voorzien. De bluetooth module is op de zender en ontvanger aangesloten van de Arduino. Afhankeljk van deze data gaan er uitgangen hoog of laag(pin 2 t.e.m.5), naargelang deze 4 signalen zal de motor driver één, of allebij de motoren aansturen voor voor of achteruit te gaan.
